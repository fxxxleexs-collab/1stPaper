# 巴赫早中期风格迁移研究：实验设计与结果分析方案

## 1. 研究目标

本研究旨在验证一套“符号音乐特征提取 + 时空事件建模 + 风格距离分析”的计算音乐学框架，考察巴赫早中期作品是否呈现出从北德键盘/管风琴传统向意大利协奏曲风格、再向综合型器乐风格迁移的轨迹。

核心问题：

> 巴赫不同时期作品在符号音乐特征空间中的位置变化，是否能够与其生平活动轨迹、历史接触事件和外部风格影响源相互印证？

---

## 2. 数据设计

### 2.1 核心数据

* 巴赫作品 MusicXML
* 参照作曲家作品 MusicXML

  * 北德传统：Buxtehude, Böhm, Reincken
  * 意大利传统：Vivaldi, Corelli
  * 德国综合型参照：Telemann，可选 Handel

### 2.2 元数据字段

每部作品至少标注：

```text
work_id
composer
title
catalogue_no
genre
instrumentation
estimated_year_start
estimated_year_end
period_label
place_composed
latitude
longitude
historical_event_label
source_confidence
xml_path
```

### 2.3 生涯阶段建议

```text
Early / Lüneburg-Arnstadt-Mühlhausen
Weimar
Köthen
Early Leipzig
```

第一篇建议重点放在 Early、Weimar、Köthen 三个阶段。

---

## 3. Segment 切分策略

统一采用固定小节数切分。

### 3.1 基本方案

* 每 16 小节切分为一个 segment
* 不足 16 小节的尾段可保留或舍弃，需在实验中统一处理
* 对长作品按 movement 或 section 分层处理

### 3.2 稳健性检验

可额外测试：

```text
8 bars
16 bars
32 bars
```

若不同切分尺度下主要结论一致，则说明结果具有稳健性。

### 3.3 分析单位

采用两级结构：

```text
segment-level features
→ work-level aggregation
```

作品层面的聚合方式包括：

```text
mean
median
standard deviation
feature entropy
segment cluster distribution
distance to reference centroid
```

---

## 4. 表征与特征提取设计

本研究采用两类表征：

```text
MusicBERT embedding
+
white-box symbolic features
```

MusicBERT 用于捕捉整体符号音乐表征；白盒特征用于音乐学解释、可视化和消融实验。

---

# 5. 七大白盒特征家族

## 5.1 音高特征 Pitch Features

用于描述作品的整体音高空间和旋律高度分布。

代表特征：

```text
pitch_range
mean_pitch
pitch_std
pitch_class_histogram
scale_degree_distribution
climax_position
phrase_final_pitch_distribution
pitch_entropy
```

可解释问题：

* 不同时期作品的音域是否扩大或收缩？
* 巴赫是否在某些阶段使用更集中的音高区域？
* 旋律终止音是否呈现更稳定的调式/调性倾向？

---

## 5.2 音程特征 Interval Features

用于描述旋律运动方式。

代表特征：

```text
melodic_interval_histogram
stepwise_motion_ratio
leap_ratio
large_leap_ratio
repeated_note_ratio
direction_change_rate
interval_entropy
interval_contour_distribution
```

可解释问题：

* 早期作品是否具有更自由、更跳跃的幻想性线条？
* 魏玛或克滕时期是否表现出更规整的级进和模进性？
* 巴赫与北德传统或意大利协奏曲传统的旋律运动是否接近？

---

## 5.3 节奏特征 Rhythm Features

用于描述作品的节奏密度、规整性与运动感。

代表特征：

```text
note_duration_histogram
mean_note_duration
duration_entropy
onset_density
rest_ratio
syncopation_index
dotted_rhythm_ratio
rhythmic_regularitiy
rhythmic_ngram_distribution
```

可解释问题：

* 魏玛以后是否出现更强的 motor rhythm？
* 意大利协奏曲风格是否体现为更高的节奏规整性？
* 法国式或舞曲性特征是否能通过附点节奏、节奏型分布体现？

---

## 5.4 调性与和声特征 Tonal-Harmonic Features

用于描述调性组织、半音化和局部调性变化。

代表特征：

```text
key_signature
estimated_key
major_minor_mode
chromaticism_ratio
non_diatonic_pitch_ratio
modulation_count
modulation_density
local_key_diversity
cadence_like_pattern_count
circle_of_fifths_motion
```

可解释问题：

* 意大利协奏曲影响是否表现为更明确的调性区域？
* 早期幻想性作品是否具有更高的局部调性游移？
* 克滕时期器乐作品是否表现出更清晰的调性规划？

---

## 5.5 复调与织体特征 Polyphony / Texture Features

用于描述声部组织、纵向结构和对位密度。

代表特征：

```text
active_voice_count
voice_density
simultaneity_density
voice_crossing_count
vertical_interval_distribution
consonance_dissonance_ratio
contrapuntal_independence
imitation_density_proxy
homophony_polyphony_ratio
```

可解释问题：

* 巴赫早期作品是否更接近北德管风琴复调传统？
* 不同时期作品的声部密度是否发生变化？
* 风格迁移是否不仅体现在旋律/节奏层面，也体现在声部组织方式上？

---

## 5.6 动机与模进特征 Motif / Sequence Features

用于捕捉局部语汇、动机复现和结构性重复。

代表特征：

```text
melodic_ngram_frequency
interval_ngram_frequency
rhythmic_ngram_frequency
motif_recurrence_rate
sequence_density
transposed_repetition_count
inversion_like_pattern_count
self_similarity_statistics
compression_ratio
```

可解释问题：

* 魏玛以后是否出现更强的模进组织？
* 巴赫作品是否在某些阶段更“图式化”或更“可压缩”？
* 意大利协奏曲风格是否通过高频模进和局部重复进入巴赫写作？

---

## 5.7 无监督几何补充特征 Unsupervised Geometric Features

用于自动发现高频局部结构，作为白盒特征的补充。

推荐方法：

```text
interval n-gram mining
rhythm n-gram mining
interval-rhythm combined pattern mining
TF-IDF distinctive pattern extraction
self-similarity matrix
segment clustering
frequent contour pattern discovery
```

使用原则：

* 不直接声称“自动发现真正动机”
* 更稳妥地称为 recurrent symbolic patterns 或 frequent melodic-rhythmic cells
* 用于辅助解释不同阶段、不同作曲家的局部语汇差异

可解释问题：

* 哪些局部音型最能区分 Bach Early 与 Bach Weimar？
* 哪些 pattern 使 Bach Weimar 更接近 Vivaldi？
* 哪些 recurrent cells 体现了北德传统或意大利协奏曲风格？

# 6. 特征家族可视化与消融实验

## 6.1 特征家族可视化

每个特征家族单独降维或可视化：

```text
Pitch-only UMAP
Interval-only UMAP
Rhythm-only UMAP
Tonal-only UMAP
Polyphony-only UMAP
Motif/Sequence-only UMAP
Geometric-only UMAP
All-features UMAP
MusicBERT-only UMAP
MusicBERT + white-box features UMAP
```

目的：

* 观察哪些特征家族最能区分时期或作曲家
* 判断风格迁移主要发生在哪些音乐维度
* 避免所有特征混合后无法解释

## 6.2 消融实验

逐一移除某个特征家族：

```text
All features
- Pitch
- Interval
- Rhythm
- Tonal
- Polyphony
- Motif/Sequence
- Geometric
```

观察指标：

```text
period classification accuracy
composer classification accuracy
distance curve stability
cluster purity
silhouette score
reference centroid distance
```

核心目的：

> 判断哪类特征对恢复巴赫风格迁移轨迹最关键。

# 7. 时空建模设计

## 7.1 基本坐标

采用简单时空坐标：

```text
x = longitude
y = latitude
t = estimated year or period midpoint
```

不在第一篇中引入复杂马尔科夫模型。

## 7.2 Event Window

围绕关键历史事件建立事件窗口。

建议事件：

```text
1705/06 Lübeck visit
1708 Weimar appointment
1713/14 Italian concerto contact / transcription period
1717 Köthen appointment
1723 Leipzig appointment
```

基本分析方式：

```text
before_event
after_event
event_window
```

比较事件前后：

```text
feature shift
distance shift
embedding shift
segment type distribution shift
```

## 7.3 Distance Curve

构造巴赫不同时期作品到参照风格中心的距离曲线。

参照中心：

```text
North German centroid = Buxtehude + Böhm + Reincken
Italian centroid = Vivaldi + Corelli
German mixed centroid = Telemann / Handel
```

核心指标：

```text
Distance(Bach_t, NorthGerman_centroid)
Distance(Bach_t, Italian_centroid)
Distance(Bach_t, GermanMixed_centroid)
```

也可构造风格吸引指数：

```text
Italianization Index
= Similarity(Bach_t, Italian centroid)
- Similarity(Bach_t, North German centroid)

North Germanity Index
= Similarity(Bach_t, North German centroid)
- Similarity(Bach_t, Italian centroid)

Synthesis Index
= balanced similarity to multiple style centroids
```

# 8. 结果分析与验证框架

最终结果分析不应只是展示聚类图，而应形成一条证据链：

```text
整体风格空间
→ 时间轨迹
→ 参照距离曲线
→ 特征贡献
→ segment-level 风格混合
→ 历史事件窗口检验
→ 风格迁移机制解释
```

---

## 8.1 分析角度一：整体风格空间

### 目标

回答：

> 巴赫作品整体位于哪些参照风格之间？

### 方法

* PCA / UMAP / t-SNE
* composer-level centroid
* genre-level centroid
* confidence ellipse
* cluster visualization

### 观察重点

* 巴赫是否靠近北德传统、意大利传统或综合型德国风格？
* 巴赫是否位于多个风格中心之间？
* 巴赫是否表现出“中介性”或“综合性”位置？

### 可能结论

> 巴赫作品并不稳定归属于单一风格中心，而是在北德、意大利和德国综合型风格之间形成过渡区域。

---

## 8.2 分析角度二：时间轨迹

### 目标

回答：

> 巴赫不同时期的风格中心是否随时间发生方向性移动？

### 方法

* period centroid trajectory
* time-colored UMAP
* period-wise embedding centroid
* trajectory line in reduced feature space

### 观察重点

* Early 阶段是否靠近北德参照？
* Weimar 阶段是否向意大利参照移动？
* Köthen 阶段是否形成器乐综合区域？
* Leipzig early 是否出现新的体裁化或制度化分化？

### 可能结论

> 巴赫早中期风格不是随机波动，而是在特征空间中呈现与生涯阶段相对应的方向性迁移。

---

## 8.3 分析角度三：参照距离曲线

### 目标

回答：

> 巴赫在不同时期分别接近哪些外部风格源？

### 方法

计算：

```text
Distance(Bach_period, NorthGerman_centroid)
Distance(Bach_period, Italian_centroid)
Distance(Bach_period, GermanMixed_centroid)
```

可视化：

```text
distance curve
similarity curve
heatmap
attraction index curve
```

### 观察重点

* 早期是否更接近 Buxtehude/Böhm/Reincken？
* 魏玛时期是否与 Vivaldi/Corelli 距离缩小？
* 克滕时期是否不再单纯意大利化，而是靠近多个风格中心？

### 可能结论

> 魏玛时期巴赫与意大利协奏曲参照的距离缩小，但克滕时期更体现为多源风格综合，而非单一模仿。

---

## 8.4 分析角度四：特征贡献分析

### 目标

回答：

> 风格迁移具体发生在哪些音乐维度？

### 方法

* feature importance
* permutation importance
* SHAP，可选
* standardized mean difference
* effect size plot
* family-level ablation
* family-level visualization

### 观察特征家族

```text
Pitch
Interval
Rhythm
Tonal
Polyphony
Motif/Sequence
Geometric
```

### 观察重点

* 意大利化主要由节奏规整性、模进密度、调性组织推动吗？
* 北德性主要由复调密度、声部组织、幻想性音程结构推动吗？
* 哪些特征家族在消融后导致风格距离曲线不稳定？

### 可能结论

> 巴赫风格迁移并非单纯音高变化，而是由节奏、模进、调性组织与织体结构共同推动。

---

## 8.5 分析角度五：Segment-level 风格混合

### 目标

回答：

> 风格迁移是整部作品整体替换，还是作品内部不同语汇比例的变化？

### 方法

* segment clustering
* segment classification
* segment type distribution
* work-level segment aggregation
* stacked bar chart
* segment proportion comparison

### Segment 类型示例

```text
North German-like segment
Italian-like segment
Contrapuntal dense segment
Dance-like segment
Mixed segment
```

### 观察重点

* Early 作品是否包含更高比例的 North German-like segments？
* Weimar 作品是否出现更多 Italian-like segments？
* Köthen 作品是否呈现更高的 mixed segment ratio？
* 单部作品内部是否存在风格混合？

### 可能结论

> 巴赫风格迁移不是从一种风格整体替换为另一种风格，而是不同局部语汇在作品内部比例的重新分配。

---

## 8.6 分析角度六：历史事件窗口检验

### 目标

回答：

> 风格变化是否发生在合理的历史事件节点附近？

### 方法

围绕事件建立 before / after 比较：

```text
before Lübeck vs after Lübeck
before Weimar vs after Weimar
before Italian concerto contact vs after contact
before Köthen vs after Köthen
```

统计方法：

```text
effect size
Mann-Whitney U test
permutation test
bootstrap confidence interval
mixed-effects model
```

### 观察重点

* 1705/06 吕贝克之行后，北德特征是否增强？
* 1708 魏玛任职后，作品是否出现意大利化趋势？
* 1713/14 意大利协奏曲接触前后，与 Vivaldi/Corelli 的距离是否变化？
* 1717 克滕任职后，器乐综合风格是否增强？

### 可能结论

> 风格空间中的变化并非任意出现，而是与巴赫关键活动节点和已知音乐史事件具有时间上的对应关系。

---

## 8.7 分析角度七：风格迁移机制解释

### 目标

回答：

> 巴赫早中期风格迁移的机制是什么？

### 方法

综合前六类结果，构建解释模型。

建议机制模型：

```text
北德传统吸收
→ 意大利协奏曲语言进入
→ 器乐综合风格形成
```

### 阶段一：北德传统吸收

对应：

```text
Lüneburg / Arnstadt / Lübeck
```

可能表现：

```text
higher similarity to North German centroid
higher polyphonic density
more fantasy-like local patterns
greater textural variability
```

### 阶段二：意大利协奏曲语言进入

对应：

```text
Weimar
```

可能表现：

```text
higher similarity to Italian centroid
higher rhythmic regularity
higher sequence density
clearer tonal organization
more Italian-like segments
```

### 阶段三：器乐综合风格形成

对应：

```text
Köthen
```

可能表现：

```text
balanced similarity to multiple centroids
higher segment diversity
stronger motivic organization
increased mixed-style segment ratio
```

### 最终解释

> 巴赫早中期风格迁移应理解为“外部风格输入 — 局部语汇吸收 — 体裁化综合”的连续过程，而不是从一种风格向另一种风格的简单线性替换。

# 10. 最终判断标准

本研究的成功不取决于聚类图是否“好看”，而取决于能否回答四个问题：

## 10.1 方向性

巴赫作品是否随时间向某些参照风格发生可解释的移动？

## 10.2 节点性

这种移动是否出现在合理的历史事件窗口附近？

## 10.3 维度性

变化主要发生在音高、音程、节奏、调性、复调、动机/模进还是无监督几何模式上？

## 10.4 机制性

这种变化是风格替换、风格叠加、局部吸收，还是体裁化综合？

---

# 11. 推荐主结论表述

本研究可将最终结论组织为：

> 巴赫早中期作品的风格演化并非线性地从一种风格转向另一种风格，而是表现为一种多源语汇的阶段性重组。早期作品在符号音乐特征空间中更接近北德键盘与管风琴传统；魏玛时期作品与意大利协奏曲参照的距离显著缩小，尤其体现在节奏规整性、模进密度和调性组织特征上；克滕时期则不再表现为单一的意大利化，而是在多个参照风格之间形成更高程度的混合结构。由此可见，巴赫早中期风格迁移可被理解为“外部风格输入 — 局部语汇吸收 — 体裁化综合”的连续过程。
